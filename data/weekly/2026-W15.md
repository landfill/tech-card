Top 5 하이라이트
1. **System Card: Claude Mythos Preview [pdf]** — 등장 6회. Anthropic이 공개한 Mythos 시스템 카드는 모델의 적응적 사고(adaptive thinking), 멀티모달 입력 처리와 사이버보안·자율 코딩 역량을 강조하며 연구용 게이티드 프리뷰로 배포되었다. 문서와 후속 보도는 모델의 능력과 한계를 함께 제시하면서 보안·배포 정책 논쟁을 촉발했다.
2. **Issue: Claude Code is unusable for complex engineering tasks with Feb updates** — 등장 4회. 커뮤니티 보고와 토론에서는 2월 이후 Claude Code의 품질 저하(예: 'thinking depth' 감소, 복잡한 엔지니어링 작업에서의 실패 사례)가 반복적으로 지적되었고, 이로 인해 에이전트 신뢰성·디버깅 비용 문제가 재조명되었다.
3. **Announcing the OpenAI Safety Fellowship** — 등장 2회. OpenAI는 독립적 안전·정렬 연구를 지원하기 위한 Safety Fellowship을 발표했으며, 이는 안전 연구 생태계 확대와 차세대 인재 육성 의도를 담고 있다(공식 페이지 참조).
4. **Introducing Claude Managed Agents, now in public beta.** — 등장 4회. Anthropic은 에이전트 오케스트레이션·권한·샌드박싱을 포함한 Managed Agents를 퍼블릭 베타로 내놓아 에이전트의 프로덕션 전환을 단축하려는 제품화 흐름을 보였다; 이 발표는 대규모 배포·운영 기능을 서비스 차원에서 제공하려는 방향을 의미한다.
5. **@saravanankalya4: GitHub Copilot Chat’s CamoLeak (CVE-2025-59145, CVSS 9.6) ...** — 등장 3회. 해당 보고는 간접적 프롬프트 인젝션이 데이터 유출로 이어질 수 있음을 보여주는 사례로 주목되었고, Copilot 계열의 실무적 보안·신뢰성 문제 논의와 맞물려 회자되었다.
